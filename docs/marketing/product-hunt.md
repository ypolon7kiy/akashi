# Product Hunt Launch Draft

## Tagline (under 60 chars)

AI agent rules IDE + session analytics for VS Code

## Description

Akashi brings two things to your editor: (1) a unified view of all your AI
agent rules -- CLAUDE.md, .cursor/rules, AGENTS.md, GEMINI.md -- with an
interactive D3 graph and community add-ons marketplace, and (2) Pulse, a
Claude Code session analytics dashboard with conversation replay,
tool-execution Gantt charts, subagent trees, and activity heatmaps.
See what your AI agents actually do.

## Maker Comment (draft)

Hey Product Hunt! I'm Yura, the maker of Akashi.

I built this because I use Claude Code, Cursor, and Gemini across different
projects, and my AI rules were scattered everywhere. When they conflicted,
agents did unexpected things -- and I had no way to see what happened
during long sessions.

Akashi solves both problems:

1. **Rules management** -- every guideline file across 4 AI tools, unified in
   one sidebar with an interactive D3 graph, search, and a community
   add-ons marketplace.

2. **Pulse analytics** -- reads your Claude Code session data and shows
   conversation replay, tool-execution Gantt charts (see parallelism and
   bottlenecks!), subagent trees, and activity heatmaps.

It's fully open source (Apache 2.0) and works in VS Code and Cursor.
Would love your feedback -- especially on what analytics you'd want to
see next!
